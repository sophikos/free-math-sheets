# Free Math Sheets

[Free Math Sheets](https://www.freemathsheets.com) is a web application that generates math worksheet PDFs for practicing different math topics. 

This project only has a frontend. There is no backend as no user data it stored. The PDFs are generated using frontend tools.

## How to generate Math Worksheets with Free Math Sheets

The left hand side has an editor.

You decide the title of the worksheet.

Then you pick a grade level, category, skill, etc. and the number of problems
you want. Finally click "Generate Worksheet" and you will see a PDF displayed.
You can download it and there is also an Answer Key provide that you can 
download.

This project is mobile friendly as well.

## How to run locally

Clone the repo then in the command line go the repo location.

```bash
npm run res:build
npm run dev
```

Now it is hosted in localhost and you can access it in your web browser.
